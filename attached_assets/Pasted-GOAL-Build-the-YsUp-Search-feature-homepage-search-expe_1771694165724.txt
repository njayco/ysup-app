API ROUTES
1) GET /api/books?q=
- Calls Google Books volumes API
- Normalizes data to: id, title, authors[], publishedDate, publisher, thumbnail, previewLink, snippet

2) GET /api/wiki?q=
- Calls Wikipedia search API
- Normalizes data to: id (title or pageid), title, snippet/extract, url

3) POST /api/summarize
Body: { query: string, source: "books"|"wiki", item: {id,title,snippet?,authors?,publisher?,publishedDate?,url?} }
- Checks Prisma cache table first:
  - key = hash(query + source + item.id)
- If cached, return cached summary
- If not cached, call OpenAI and store summary.
